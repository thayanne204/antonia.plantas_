# Antônia Plantas

🌿 Projeto Antonia Plantas: Landing Page High-End
Este projeto consiste no desenvolvimento de uma vitrine digital sofisticada para a marca Antonia Plantas, especializada em Rosas do Deserto. O objetivo principal foi elevar o posicionamento de mercado da marca através de uma interface de alto luxo, unindo estética minimalista e performance técnica.

💎 Diferenciais do Projeto
Design Dark Lux: Implementação de uma interface em Dark Mode para maximizar o contraste cromático e a vivacidade das plantas, utilizando uma paleta baseada em Preto Profundo, Ouro Nobre e Bordô Clássico.

Tipografia Curada: Uso estratégico da fonte Cinzel para títulos, evocando elegância clássica e autoridade, equilibrada com Montserrat para uma leitura fluida e moderna no corpo do texto.

Arquitetura Responsiva: Construção realizada com CSS Grid e Flexbox, garantindo que a experiência visual seja impecável tanto em dispositivos móveis quanto em monitores de ultra-resolução.

Foco em Conversão (CRO): Desenvolvimento focado na jornada do usuário, com Call-to-Actions (CTAs) estratégicos e integração direta com canais de atendimento (WhatsApp).

🛠️ Tecnologias Utilizadas:

HTML5 Semântico.

CSS3 Avançado (Variáveis, Grid, Flexbox, Animações).

Google Fonts para tipografia personalizada.

Metodologia Mobile-First.

📈 Desafio de Engenharia de Software
O maior desafio deste projeto foi equilibrar o alto impacto visual com a performance de carregamento. No 5º período de Engenharia de Software, o foco foi aplicar conceitos de Design Atômico e Psicologia das Cores para transformar um catálogo de produtos em uma experiência de marca exclusiva para o cliente real.
